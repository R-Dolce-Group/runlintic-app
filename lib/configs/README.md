# `@rdolcegroup/eslint-config`

Collection of internal shared ESLint configurations for Runlintic projects.

## Usage

```js
// For base TypeScript/Node.js projects
import { config } from '@rdolcegroup/eslint-config/base';

// For React projects
import { config } from '@rdolcegroup/eslint-config/react-internal';

// For Next.js projects
import { nextJsConfig } from '@rdolcegroup/eslint-config/next-js';
```
